## 1. 本批次目标

- 支持 **真实实时检索** 类需求（天气、新闻/百科等），通过 **T2 可配置内置连接器** 落地；远期 **T3 MCP**。
- 提供 **能力库 UI**（列表、创建/编辑能力）与 **Agent 侧能力多选**（无能力时引导创建）。
- 明确 **T1～T3** 类型与 **对应 UI**（**不含 T4 高级编辑器**），避免未接线工具进入可选列表。

## 3. 验收要点（增量）

1. 用户可在 **能力库** 中创建、编辑、删除（或停用）至少一种 **T2** 能力实例；密钥不落明文表。
2. 用户可在 **创建/编辑 Agent** 中为助手勾选能力；无能力时可跳转到创建能力。
3. 对话中模型可 **实际发起 tool 调用** 并返回 **与外部 API 一致** 的实时结果（验收用例：天气或检索类至少一条打通）。
4. 未实现 handler 的能力 **不可选** 或 **灰显+说明**。
